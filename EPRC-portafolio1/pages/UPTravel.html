UPTravel: Aplicación de Viajes

Descripción

Este proyecto se realizó en el tercer semestre de la carrera como proyecto final de la clase: Programación Avanzada. En este proyecto teníamos libertad total sobre el tipo de aplicación que podíamos hacer pero debíamos implementar todo lo aprendido a lo largo del semestre. Con esto en mente, y luego de considerar varias sugerencias que el profesor dio a la clase, mi equipo y yo decidimos hacer esta sencilla aplicación de viajes: UPTravel.

Aprendí como utilizar el lenguaje C++ y muchas de las funciones con las que cuenta este lenguaje. También descubrí que tan sencillo puede ser desarrollar aplicaciones del estilo de UPTravel así como diseñar la interfaz gráfica para diversas aplicaciones.

El código fue desarrollado en Visual Studio con el lenguaje C++ y aquí si se podían almacenar y editar datos para un viaje y los integrantes de dicho viaje. Este sencillo programa efectivamente se podía utilizar día a día si no se tiene inconveniente en no contar con una interfaz gráfica. A pesar de ello es un programa bastante intuitivo, con instrucciones claras y sencillo de usar.
